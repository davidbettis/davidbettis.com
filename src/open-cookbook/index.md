## Summary

Open Cookbook is an iOS app that lets you organize, search, and view your recipe library on your own terms without being locked into a proprietary ecosystem. Recipes are stored on your personal device or your iCloud drive. Recipes are formatted in [RecipeMD](https://recipemd.org/), an open format based on markdown. Build a recipe library you can pass down to your kids.

Available for download at the Apple app store today.

## FAQs

### Why would I want to use this app?

There are a ton of recipe apps out there. Based on my research, it seems like most of these apps store recipes in a properietary database. I wanted an app that stored its data in an open format, so it will be relevant in the distant future (think 10+ years).

### What's this about a Claude API token to import recipes?

If you want to import recipes by website or by photo, you need a Claude API token. Claude is really good at understanding websites and photos. However, it is too expensive for me to provide to you for free. You will need to create a Claude account. Add some funds and then you'll pay 1-2 cents per import. If all this seems cumbersome, we are currently working on being able to import recipes, strictly relying on your phone's processor. Stay tuned. Until then, please see these directions to setup a Claude account and add it to your Open Cookbook app.

1. Go to [platform.claude.com](https://platform.claude.com/) and create an account.
2. Navigate to Manage > API keys
3. Click on Billing
4. Click on "Buy Credits" I suggest adding $5 to start.
3. After the purchase is complete, click on API keys
4. Click on "+ Create Key"
5. Name the key "open-cookbook-api-key" and create it
6. It will display the contents of the key. After you close that dialog, you can never see it again! Save it in secure place, like your password manager.
7. In the Open Cookbook app, go to **Settings**, go to the Import Recipe section, and paste your API key into the **Claude API Key** field.
